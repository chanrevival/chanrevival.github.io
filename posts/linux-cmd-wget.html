wget命令

命令用途

wget 是一个用于非交互式网络文件下载的 Linux 命令行工具，支持 HTTP、HTTPS 和 FTP 协议，能够递归下载文件或镜像整个网站。

常用用法示例

  1. 下载单个文件

    从指定 URL 下载文件到当前目录。

  2. 指定输出文件名

    通过 -O 参数将下载文件重命名为 backup.zip。

  3. 后台下载

    使用 -b 参数让下载任务在后台运行。

  4. 断点续传

    通过 -c 参数恢复中断的下载任务。

  5. 递归下载整个网站

    -r 启用递归下载，-l 2 限制递归深度为 2 层。

  6. 限制下载速度

    通过 –limit-rate 将下载速度限制为 100KB/s。

  7. 批量下载多个文件

    通过 -i 参数从 url-list.txt 文件中读取多个 URL 进行批量下载。

  8. 下载需要认证的文件

    使用 –user 和 –password 参数提供 HTTP 认证凭据。

常用参数选项

  • -O <文件>
    指定输出文件名，避免使用 URL 中的默认文件名。
  • -b, –background
    后台模式运行，适合大文件下载。
  • -c, –continue
    断点续传，恢复未完成的下载。
  • -r, –recursive
    递归下载，用于抓取整个目录或网站。
  • –limit-rate=<速度>
    限制下载速度（如 100k 或 1m）。
  • -i <文件>
    从文本文件中读取多个 URL 进行批量下载。
  • –user=<用户名>, –password=<密码>
    提供 HTTP/FTP 认证的用户名和密码。
  • -q, –quiet
    静默模式，不输出下载过程信息。
